\# 100 Days of DSA



This repository contains my daily Data Structures and Algorithms practice implemented in C as part of a 100-day consistency challenge.



\## Structure

* Solutions are grouped in folders of 10 days
* Each file contains one clean implementation



\## Language

* C (GCC compatible)



\## Goal

To build strong fundamentals in Data Structures and Algorithms.



\##Topic covered:

1. Array insertion
2. Array deletion
3. linear search and comparison counting
4. Two-pointer approach
5. Merge sorted array
6. Remove duplicates
7. recursive Fibonacci
8. recursion power
9. mirror string
10. palindrome check
11. Matrix Addition
12. Symmetric Matrix
13. Boundary to Center Matrix Walk
14. Check Identity Matrix
15. Diagonal Sum of Matrix
16. Frequency counter for Array
17. Maximum and Minimum Element in Array
